# Logos

Two variants, each with icon-only, horizontal wordmark, and stacked wordmark — following [CNCF artwork conventions](https://github.com/cncf/artwork).

All logos use exact SVG path data from official Kubernetes (CNCF) and Claude (Anthropic) sources. All orange (`#D97757`) with white shield interior.

## Variant A: Claude Asterisk

| Type | File |
|------|------|
| Icon | [k8laude-icon-color.svg](asterisk/k8laude-icon-color.svg) |
| Horizontal | [k8laude-horizontal-color.svg](asterisk/k8laude-horizontal-color.svg) |
| Stacked | [k8laude-stacked-color.svg](asterisk/k8laude-stacked-color.svg) |

Best for: general branding, favicon, documentation.

- **Favicon/app icon**: asterisk icon variant
- **Banner**: asterisk icon (inline SVG data URI in all page banners)
